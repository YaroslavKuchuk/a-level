### Ковариантность

var asteroid = new Asteroid();
 
ICollidable collidable = asteroid;

Ковариантность - совместимость назначения (assignment compatibility).

### Значения по умолчанию

T _field = default(T);

* Ссылочные типы – null
* Значимые – 0

* ArrayList => List<T>

* HashTable => Dictionary<T, U>

* Stack => Stack<T>

* Queue => Queue<T>
